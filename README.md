# gulp4-project

>Сборка работает на gulp версии 4.0.

## npm
1. ```clone this repo```
2. ```cd path/to/...```
3. ```npm i```
4. ```npm install --global gulp-cli```
5. ```npm i gulp --save-dev```
6. ```gulp```

## yarn
1. ```clone this repo```
2. ```cd path/to/...```
3. ```yarn install```
4. ```yarn add global gulp-cli```
5. ```yarn add gulp -D```
6. ```gulp```
